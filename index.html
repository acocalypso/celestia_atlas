About this atlas

This standalone shell uses the same framework-neutral Celestia Atlas viewer as embedded applications. Stars, constellations, moving objects, reference layers, landscapes, and deep-sky objects therefore share one rendering implementation. It has no Aladin, Stellarium Web, external fonts, or online catalogue calls. At narrow fields, the optional DSS2 Color survey progressively adds real photographic sky imagery from a CDS-listed MAST mirror.

Data scope

The public build contains OpenNGC, all 86 Abell-1966 planetary nebulae, Abell/ACO galaxy clusters, the LDN, Barnard, LBN, Sharpless, vdB, and RCW groups, plus a HYG star layer through visual magnitude 6.5. Catalogue coordinates are normalized to ICRS. Stars use their available B-V colours, while galaxies and nebulae use type-specific symbols and scaled catalogue footprints. Faint objects appear progressively as you zoom in, while every bundled entry remains searchable offline. The Milky Way panorama and available local DSO images require no runtime network access. DSS tiles are requested on demand, may be cached after viewing, and fall back to the local panorama when a field is unavailable offline.

Controls

Drag to pan, wheel or pinch to zoom, click an object for details, and use /, G, C, D, L, H, N, R, or F.

Offline installation

Serve this folder once on localhost and choose “Install app” in a compatible browser. The included service worker caches every required atlas file and can retain a bounded set of DSS tiles you have viewed. Unseen DSS fields still need a connection; the local atlas remains usable without them.
