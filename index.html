About this atlas

This build uses a custom native Canvas celestial-sphere renderer. It has no Aladin, Stellarium Web, Three.js, external fonts, remote imagery, or online catalog calls.

Data scope

The compact catalog is designed for naked-eye navigation and popular telescope targets. Coordinates are J2000. The Milky Way overlay is a calculated galactic-coordinate plane, not a photographic image. Object sketches are schematic.

Controls

Drag to pan, wheel or pinch to zoom, click an object for details, and use /, G, C, D, L, H, N, R, or F.

Offline installation

Serve this folder once on localhost and choose “Install app” in a compatible browser. The included service worker caches every required file.
